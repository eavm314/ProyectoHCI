## Instrucciones para correr el proyecto

1. Instalar _node.js_ de la pagina https://nodejs.org/en

2. Clonar el repositorio con
``
git clone
``

3. Instalar las herramientas necesarias con
``
npm install
``

4. Ejecutar el proyecto con el comando
``
npm run dev
``
